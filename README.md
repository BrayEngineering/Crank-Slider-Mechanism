# Crank Slider Mechanism

This project contains CAD models of parts used in a crank-slider mechanism.

## Parts Layout

![Parts Layout](https://github.com/user-attachments/assets/99c83fe2-85aa-4eeb-aff0-9443b0e326ad)

## Parts Modeled

- Crank disk
- Connecting rods
- Slider block
- Base component

## Files Included

- CS-Crank.ipt
- CS-Rod.ipt
- CS-Slider.ipt
- CS-Base.ipt

## Skills Demonstrated

- 3D CAD modeling
- Mechanical component design
- Understanding of crank-slider motion systems
